# Machine learning based classification of depression

Repository of codes for the machine learning based based classification of
depression

# Usage

Use Jupyter notebooks (.ipynb files) to load data, transform data, and explore
models.

Use scripts in grid_search directory to batch run combinations of data, and
model configurations.

# Prerequisites

+ Python 3.6+
+ Scikit learn
+ Jupyter notebook
+ Numpy
+ Pandas
+ Scipy
+ Matplotlib
+ Seaborn
+ XGBoost
+ mlxtend

# List of directories

+ data

Data and results.

+ grid_search

Script to perform batch grid search. Script to parse results.
